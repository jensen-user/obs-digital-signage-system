# Default Slideshow

This folder contains content that will be displayed during all other operating hours (when the PC is running, outside Sunday service times).

## Schedule
- **Active**: All times except Sunday 08:00-13:30
- **Transition**: Fade (smooth transition suitable for general content)

## Content Guidelines
- Add images with numbered prefixes: `01_general_info.jpg`, `02_event_promo.png`, etc.
- Add videos: `03_community_news.mp4`, `04_weekly_schedule.mp4`, etc.
- Supported formats:
  - Images: JPG, PNG, BMP, GIF, TIFF, WebP
  - Videos: MP4, MOV, AVI, MKV, WMV, WebM

## Example Files
- `01_welcome.jpg` - General welcome slide
- `02_weekly_events.png` - Weekly event schedule
- `03_contact_info.jpg` - Contact information
- `04_community_news.mp4` - Video community updates

Files are displayed in alphabetical order based on filename.
