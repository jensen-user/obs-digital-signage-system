# Sunday Service Slideshow

This folder contains content that will be displayed during Sunday service hours (08:00-13:30).

## Schedule
- **Active**: Sundays, 08:00 - 13:30
- **Transition**: Stinger Transition (dramatic transition suitable for service content)

## Content Guidelines
- Add images with numbered prefixes: `01_welcome.jpg`, `02_announcement.png`, etc.
- Add videos: `03_sermon_intro.mp4`, `04_event_promo.mp4`, etc.
- Supported formats:
  - Images: JPG, PNG, BMP, GIF, TIFF, WebP
  - Videos: MP4, MOV, AVI, MKV, WMV, WebM

## Example Files
- `01_welcome_to_service.jpg` - Welcome slide
- `02_todays_sermon.png` - Sermon topic
- `03_prayer_slide.jpg` - Prayer slide
- `04_announcements.mp4` - Video announcements
- `05_closing.jpg` - Closing slide

Files are displayed in alphabetical order based on filename.
